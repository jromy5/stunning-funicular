Request a new git repository

Here you can request a new git repository. To request the main repository, $project.git, simply leave the sub-repo field blank.
MAKE SURE THE NOTIFICATION LIST AND REPO NAME ARE BOTH CORRECT!.
Incubator podling repos should be named as incubator-$podling[-$subname].git
See https://reference.apache.org/pmc/github for details on GitHub integration, which is enabled by default.
