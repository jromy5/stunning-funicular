Welcome to the Apache Software Foundation Subversion Server

The Apache Infrastructure Team provides version control for all of of the ASF projects.

You can learn more about how the ASF uses version control on our main website.

You might also be looking for the Apache Subversion Project, which can be found at subversion.apache.org

You are currently using svn.us.apache.org. This is normally selected automatically based on GeoDNS, for the best experience, please use svn.apache.org, as this DNS name will always be the most reliable.

The following geo-located mirrors are available:

  • svn.us.apache.org
  • svn.eu.apache.org

Please contact users@infra.apache.org if you have any issues with the Apache Subversion repositories.
